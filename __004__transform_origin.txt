
.element {
  transform-origin: left;
}

.element:hover {
  transform: scale(.7);
}

Все изменения с элементом будут происходить относительно левой стороны
В данном примере элемент уменьшится в размере но стартовая точка элемента будет с лева
по умолчанию стартовая точка у элемента находится в центре элемента.
